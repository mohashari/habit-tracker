Daily check-ins, streak tracking, GitHub-style heatmap calendar, and motivational quotes.

## Setup

```bash
# Clone the repo
git clone https://github.com/mohashari/habit-tracker.git
cd habit-tracker

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Start development server
npm run dev
```
